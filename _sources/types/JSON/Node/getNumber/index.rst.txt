getNumber
=========

:doc:`/types/JSON/Node/index`::getNumber

Returns the number contained in a Number Node.

Declaration
-----------

.. code-block:: cpp

	float getNumber() const;

Parameters
----------

None.

Returns
-------

The value of a Number :doc:`/types/JSON/Node/index`.

Notes
-----

This function can only be called on a :doc:`/types/JSON/Node/index` with the Number :doc:`/types/JSON/Type/index`.
