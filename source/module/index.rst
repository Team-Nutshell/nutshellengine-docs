Module
======

Modules implement NutshellEngine's main systems. **All modules are optional**, but if a module is missing, its role will not be fulfilled.

List of Modules
---------------

- :doc:`/module/asset_loader_module/index`: Implements the **loading of assets files**.
- :doc:`/module/audio_module/index`: Implements the **audio system to play sounds**.
- :doc:`/module/graphics_module/index`: Implements the **graphics engine to render frames**.
- :doc:`/module/physics_module/index`: Implements the **physics engine and collision detection system**.
- :doc:`/module/window_module/index`: Implements the **window and input systems**.
